This simple Web Application using ASP.NET Core MVC with C# supports a 
single web page. Using jQuery/Ajax for CRUD operations create/delete.

Front-end included bootstrap,font-awesome,and javascript.


The purpose of this application is for users to check their content limits insurance estimate. They will be able
to choose from a few categories and the numeric value for each item. 
The user is able to see the total for each category and the overall total for all categories.
